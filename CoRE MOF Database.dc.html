Computation-Ready Experimental (CoRE) MOF Database.

CoRE는 "코어(core)"로 읽습니다 — pronounced "core."

A curated database of over 40,000 experimentally reported, computation-ready MOF crystal structures with machine-learned properties for high-throughput material–process screening. Each structure is solvent-removed, charge-assigned, and checked for computational readiness, so it can be used directly in molecular simulations and screening workflows. The database is continuously updated with newly reported MOFs and refined classification of computation-ready (CR) and not-computation-ready (NCR) structures.
